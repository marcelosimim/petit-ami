
# Petit Ami

## Objetivo do projeto
Esse é um trabalho final de Flutter do curso de Desenvolvimento de Sistemas com ênfase em Aplicativos Móveis do [COLTEC - UFMG](http://www.coltec.ufmg.br/coltec-ufmg/).

## Resumo do projeto
O app auxilia no estudo de francês. Os exercícios podem ser cadastrados no banco de dados e separados por unidades. O app é responsável por armazenar o progresso.

![Perfil]( https://drive.google.com/uc?export=view&id=1olXGbsQ7vdWyMNo0mxDR1kalTxZUKxvg)

Além disso, há uma página para download do material, sendo que só os que estão no nível do usuário estão disponíveis para download, sendo liberados à medida que este faça progresso.

Os exercícios seguem o seguinte formato:
- Leitura, escuta e repetição
- Leitura, escuta e resposta

### Exercício de repetição
Há um áudio relacionado ao exercício e uma resposta pré-definida no banco de dados. Ao clicar no botão do microfone, o usuário pode repetir a frase que já será escutada em francês. Caso esteja certa, irá automaticamente para o próximo exercício.

![Exercício de repetição]( https://drive.google.com/uc?export=view&id=17M4Yh73ANB7bOLF9mLk_4k8D64S3rcrM)

### Exercício de resposta
Para exercícios de resposta, também há uma resposta pré-definida no banco de dados. A resposta enviada pelo usuário no banco de texto será comparada com a resposta correta. Caso essa estiver certa, seguirá para o próximo exercício.

![Exercício de resposta]( https://drive.google.com/uc?export=view&id=1--uFDgGMn3Ch7nt3hsAOPGjwBJwTaKu1)
